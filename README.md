▶️ How to Run
Install Python on your system

Clone this repository
git clone https://github.com/zeba570/dice_game.git

Open the project folder
cd dice_game
Run the program
python dice_game.py

🎮 How to Play
Enter your name
Type y to roll the dice
Type n to quit the game
Try to hit the lucky number for bonus fun!
